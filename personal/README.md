# personal/ — SMOKE Live Stack

This directory mirrors exactly what lives at `%USERPROFILE%\.openclaw\workspace` on SMOKE.

## Key folders

- `scripts/` — Epic OS kernel, media vault, trading desk, repo sync
- `bootstrap/` — Heartbeat, continuity, machine profiles (acer.json, lenovo.json)
- `skills/` — Agent skill definitions (epic-os skill + custom tools)
- `btc_epictechai_desk/` — Private trading workspace
- `epictech_ecosystem/` — Broader experiments and notes

## Important files at root (personal)

- `SESSION.md` — Current context, open loops, decisions (never committed)
- `MEMORY.md` — Long-term context (never committed)
- `CONTINUE SESSION.bat` — Primary launcher
- `Epic OS.bat` — Direct shell start

## Live paths on SMOKE

```
%USERPROFILE%\.openclaw\workspace          ← this personal/ folder
%USERPROFILE%\.openclaw\secrets            ← never in repo
OneDrive\...                               ← some desk projects
```

The entire personal stack is what powers the private `@epictech_assistant_bot` with full repo access.
